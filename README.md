# Landing Page COVID-19 Informativa

Sitio web estático informativo sobre el COVID-19. Incluye información clara y verificada sobre síntomas, medidas de prevención, vacunación, COVID largo y preguntas frecuentes, basado en guías de la OMS y organismos de salud nacionales e internacionales.

## Tecnologías

- HTML5
- CSS3
- TailwindCSS
- JavaScript

## Secciones

- ¿Qué es el COVID-19?
- Síntomas
- Cómo protegerte
- Vacunas
- COVID largo
- Preguntas frecuentes
- Fuentes verificadas
